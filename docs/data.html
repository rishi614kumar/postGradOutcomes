2  Data

2.1 Description

For this project, we are using Post-Secondary Employment Outcomes (PSEO) datasets that contain earnings and employment outcomes for graduates.

2.1.1 Data Source

The national and state-level PSEO datasets are publicly available through the United States Center for Economic Studies. Specifically, we are using the PSEOE (earnings) and PSEOF (employment rates) files from the PSEO section.

2.1.2 Data Collection

The data is collected through the Longitudinal Employer-Household Dynamics (LEHD) program. This program partners with various post-secondary institutions, state agencies, and the United States Census Bureau. LEHD merges admistrative job data with the post-secondary education records from its partners to create the PSEO datasets.

2.1.3 Data Format

We are working with CSV files downloaded directly from the public data access page. The national-level datasets contains approximately 564,000 records and 36 columns, covering aggregated earnings and employment outcomes for graduates 1 year, 5 years, and 10 years post-graduation (when available). The main identifiers in the dataset include:

  • Institution
  • Degree Level
  • Degree Field
  • Cohort Graduation Year

2.1.3.1 Mappings for Identifiers:

Please note that we enrich this data for visualization and analysis by mapping ID values to string values using external sources.

  • Institution (OPEID): Post-Secondary Institutions with OPEID
  • Degree Field (CIPCODE): CIP Code Table
  • Degree Level: NCES Degree Level Classifcations | PSEO Technical Documentation

2.1.4 Data Coverage

The data covers the years from 2001 to 2020 as of the 2023Q20 release. New data for additional partner institutions is added irregularly, such as the June 2024 update that incorporated new universities. See the PSEO data notices for their updates.

Limitations: There are regional coverage gaps in the dataset because it only includes outcomes from institutions that have partnered with the Census Bureau through the data-sharing agreements. For that reason, many states have no coverage while most others only have partial coverage. Breakdown of state-by-state coverage can be found here.
